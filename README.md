# GitHub Pages site (single page)

This folder contains a minimal single-page website for GitHub Pages.

## What to upload
- `index.html`
- `assets/style.css`
- Your PDFs in the same folder as `index.html`:
  - `YanshuaiQin_Resume_Quant_DS.pdf`
  - `YanshuaiQin_CV_Academic.pdf`

## Publish options
### Option A (recommended): User site
1) Create a repo named: `Yanshuaimath.github.io`
2) Upload these files to the root of that repo
3) Settings → Pages → Deploy from branch → `main` / root
4) Your site becomes: https://Yanshuaimath.github.io/

### Option B: Project site
Use any repo, put the files in `/docs`, then Settings → Pages → `main` / `/docs`
